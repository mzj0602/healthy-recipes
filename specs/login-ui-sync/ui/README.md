# Login UI Design — login-ui-sync

## 来源

从 Stitch MCP 获取，项目：个人健康周计划页（`projects/2553716143790562861`）
屏幕：登录页 (中文版)（`screens/3d98477367564f0ab94beb5383bea58e`）

## 设计文件

| 文件 | 说明 |
|------|------|
| `login-screen.html` | Stitch 导出的登录页 HTML 设计稿（含 Tailwind CSS） |

## 设计要点

- **尺寸**：Desktop，2560 x 2076px
- **主色调**：橙色系（`#964900` primary，`#f5821f` container），背景色 `#fdf9f3`
- **字体**：Epilogue（标题）+ Plus Jakarta Sans（正文）

## 页面结构

1. **顶部导航栏**：FreshPlate Logo（橙色）+ 返回首页链接 + Sign Up 按钮
2. **登录卡片**（居中，max-w-md）：
   - Logo 标题区：`FreshPlate` 大字 + "欢迎回来" + 副文案
   - 表单：邮箱/用户名输入框（含 person 图标）、密码输入框（含 lock 图标）
   - 选项行：记住我复选框 + 忘记密码链接
   - 登录按钮：橙色渐变（`#964900` → `#f5821f`），全宽，rounded-xl
   - 分隔线："或使用社交账号登录"
   - 社交登录：Google + 微信两栏按钮
   - 底部：立即注册链接
3. **装饰元素**：左右两侧半透明食物图片（沙拉碗、新鲜蔬菜），模糊光晕背景圆
4. **页脚**：品牌名、导航链接、版权信息
