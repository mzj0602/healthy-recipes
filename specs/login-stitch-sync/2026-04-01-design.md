### 字体方案

**引入方式**：在 `index.html` 的 `<head>` 中添加 Google Fonts `<link>` 标签。

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link href="https://fonts.googleapis.com/css2?family=Epilogue:wght@700;800&display=swap" rel="stylesheet" />
```

**使用方式**：在需要 Epilogue 的元素上使用内联 `style={{ fontFamily: 'Epilogue, serif' }}` 或 Tailwind 任意值 `font-[Epilogue]`。Plus Jakarta Sans 已在 `styles.css` 的 `:root` `font-family` 中配置，无需额外引入。

**理由**：
- `<link>` 方式是 Google Fonts 推荐的引入方式，浏览器可并行加载
- 仅加载 `wght@700;800`（Bold + ExtraBold），最小化字体文件大小
- `display=swap` 确保字体加载期间使用后备字体，不阻塞渲染
- 无需新增 npm 依赖

### 渐变按钮实现

**方案**：在 `<Button>` 组件上使用 `className` 覆盖背景色 + `style` prop 设置渐变。

```tsx
<Button
  type="submit"
  variant="default"
  size="lg"
  disabled={isSubmitting}
  className="w-full font-bold text-lg text-white py-4 rounded-[0.75rem] shadow-[0_12px_32px_rgba(28,28,24,0.06)] hover:scale-[1.02] transition-transform"
  style={{
    fontFamily: 'Epilogue, serif',
    background: 'linear-gradient(135deg, #964900 0%, #f5821f 100%)',
  }}
>
```

**理由**：
- shadcn/ui `<Button variant="default">` 默认使用 `bg-primary`，渐变无法用单一 Tailwind class 表达
- `style.background` 会覆盖 Tailwind 的 `bg-*` 类，无需自定义 variant
- `className` 中的 `rounded-[0.75rem]` 使用 Tailwind 任意值，确保圆角与需求一致（项目 `--radius` 为 `0.5rem`，`rounded-xl` 在项目中解析为 `1rem`，不符合 `0.75rem` 要求）
- 悬停效果使用 `hover:scale-[1.02]` + `transition-transform`，轻量且符合设计稿

### 输入框样式覆盖

**方案**：通过 `className` 覆盖 shadcn/ui `<Input>` 的默认边框样式。

```tsx
<Input
  className="border-none bg-[#f7f3ed] rounded-[0.75rem] py-3 pl-10 pr-4 focus:ring-2 focus:ring-[#964900]/20"
/>
```

**关键点**：
- `border-none` 移除 `<Input>` 默认的 `border border-border-input` 样式
- `bg-[#f7f3ed]` 使用 Harvest `surface-container-low` 色值
- `focus:ring-2 focus:ring-[#964900]/20` 替代默认的 `focus:ring` 样式，对齐 Harvest 主色

## 关键决策

1. **只改 login-page.tsx，不新建组件**：双列布局的 Hero 区是纯展示性 JSX，无交互逻辑，抽取为独立组件会增加文件数量但不带来复用价值。登录页是功能内聚的单页组件，保持单文件更易维护。

2. **字体通过 `index.html` `<link>` 引入**：相比 CSS `@import`，HTML `<link>` 允许浏览器更早发现字体资源并并行加载，性能更优。仅加载 Epilogue Bold/ExtraBold 两个字重，最小化传输量。Plus Jakarta Sans 已存在于项目 `font-family` 栈中，无需额外加载。

3. **渐变按钮用 `style` prop 实现**：Tailwind 无法原生表达 `linear-gradient` 背景，`style={{ background: '...' }}` 是最简洁的方式，避免新增自定义 Tailwind 插件或 CSS 类。`className` 负责其余样式（圆角、阴影、过渡），`style` 仅负责渐变背景色。

4. **Harvest 颜色用 Tailwind 任意值而非新增 CSS 变量**：本次修改仅涉及登录页，Harvest token（如 `#f7f3ed`、`#964900`）与全站现有 CSS 变量体系（`--primary: #ec7f13`）不完全一致。在登录页内用 `bg-[#f7f3ed]` 任意值精确匹配设计稿，避免修改全局 `styles.css` 引发其他页面视觉变化。

5. **Hero 图片选择 `featured-quinoa-bowl.jpg`**：该图已存在于 `public/images/`，是横版食物大图，色彩丰富，适合作为 Hero 背景。若效果不理想，可在 P4 开发时替换为其他候选图（如 `blog-meal-prep.jpg`）。
