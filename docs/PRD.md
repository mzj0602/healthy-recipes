# 产品需求文档（PRD）

**产品名称**：FreshPlate 健康菜谱平台
**版本**：v1.3
**最后更新**：2026-03-24
**技术栈**：React 18 + TypeScript + Vite + TailwindCSS

---

## 一、产品概述

FreshPlate 是一款面向健康饮食人群的 Web 应用，提供菜谱浏览、饮食计划管理和营养追踪功能。用户可以发现健康菜谱、查看详细烹饪步骤，并通过个人健康周计划管理一周饮食安排。

### 目标用户
- 关注健康饮食、希望系统管理膳食的用户
- 健身人群，需要追踪蛋白质、热量等营养指标
- 家庭用户，希望每周提前规划膳食

### 2.5 饮食计划编辑弹窗（EditPlanModal）

**v1.2（2026-03-24 新增）**

| 功能 | 描述 |
|------|------|
| 弹窗框架 | `role="dialog"`、`aria-modal="true"`、背景蒙层、× 关闭按钮 |
| 日期 Tab | 七天 Tab 栏（周一～周日），点击切换当天餐食列表，默认周一 |
| 营养目标编辑 | 编辑 `valueLabel`（文本）和 `percent`（0–100，超范围自动截断） |
| 餐食行编辑 | 编辑餐食名称（maxLength=100）和卡路里（min=0） |
| 添加餐食 | 当天列表末尾追加新行（type='其他'，title=''，calories=0） |
| 删除餐食 | 删除对应行；当天仅剩 1 条时删除按钮 disabled |
| 表单校验 | 保存时遍历全部七天：title 为空或 calories 非有效非负整数均报错 |
| 持久化 | 保存后写入 `localStorage('health-plan-data')`，刷新后恢复；格式不合法回退默认数据 |
| 取消语义 | 取消/×/蒙层均丢弃草稿，不写入任何数据 |

### 2.6 购物清单弹窗（ShoppingListModal）

**v1.3（2026-03-24 新增）**

| 功能 | 描述 |
|------|------|
| 触发入口 | 点击健康计划页"生成购物清单"按钮，弹出全屏蒙层弹窗，可通过 ×、蒙层点击或 Esc 关闭 |
| 食材来源 | 读取当前周计划（`localStorage('health-plan-data')`），通过静态映射表（`meal-ingredients-map`）将餐食名称转换为食材列表 |
| 去重合并 | 相同食材名称合并，用量累加（格式：`"150g × 3 份"`） |
| 分类展示 | 食材按五类分组：蔬果类 → 肉蛋类 → 主食类 → 调料类 → 其他，弹窗标题显示食材总数 |
| 勾选交互 | 复选框勾选后文字变灰 + 删除线；弹窗关闭后勾选状态重置 |
| 复制功能 | "复制文字"按钮将未勾选食材以纯文本写入剪贴板，显示"已复制！"提示 1.5s；不支持剪贴板 API 时自动隐藏按钮 |
| 重置勾选 | "取消全部勾选"恢复所有复选框 |
| 空状态 | 当前计划无餐食时显示空状态提示 + "去编辑计划"按钮（关闭购物清单并打开编辑弹窗） |

**验收标准**：
- [ ] 点击"生成购物清单"，弹窗在 300ms 内弹出，标题显示食材总数
- [ ] 食材按分类分组，去重合并，每条有复选框
- [ ] 勾选食材后文字变灰 + 删除线，再次点击恢复
- [ ] 点击"复制文字"，剪贴板内容为未勾选食材纯文本，页面显示"已复制！"提示
- [ ] 点击 × / 蒙层 / Esc，弹窗关闭，勾选状态重置
- [ ] 当前计划无餐食时，弹窗显示空状态，"去编辑计划"按钮可用
- [ ] 编辑计划后重新点击按钮，清单反映最新餐食数据

## 五、全局导航栏搜索（HeaderSearch）

**版本**：v1.1（2026-03-23 新增）

在导航栏右上角（个人中心按钮左侧）提供可展开/收起的全局搜索入口，用户可在任意页面快速搜索菜谱。

| 功能 | 描述 |
|------|------|
| 搜索图标 | 放大镜图标，hover 变品牌橙色 `#ec7f13` |
| 展开动画 | 点击图标，输入框以 200ms CSS transition 展开至 220px |
| 自动聚焦 | 展开后光标自动定位到输入框 |
| 关键词提交 | 按 Enter 或点击搜索按钮触发，空值不触发 |
| 跳转探索页 | 切换 `activePage` 至 `recipes`，并将关键词同步到探索页搜索栏 |
| 状态同步 | 已在探索页时直接更新搜索状态，无页面重新挂载 |
| 收起方式 | 点击输入框外部（onBlur）或按 Esc 键 |

**验收标准**：
- [ ] 任意页面下点击搜索图标，输入框在 200ms 内展开并自动聚焦
- [ ] 输入关键词按 Enter，探索页搜索栏显示相同关键词并触发筛选
- [ ] 已在探索页时搜索，菜谱列表刷新，无页面切换闪烁
- [ ] 空值按 Enter 不跳转、不触发副作用
- [ ] 点击外部或按 Esc，搜索框收起

### 2.7 轻量登录页（LoginPage）

**v1.4（2026-03-28 新增）**

| 功能 | 描述 |
|------|------|
| 默认登录入口 | 未登录打开应用时优先显示登录页，而不是直接进入首页 |
| 登录表单 | 提供用户名输入框、密码输入框、登录按钮 |
| 基础校验 | 用户名/密码为空时阻止登录并显示错误；错误凭证显示失败提示 |
| 进入主应用 | 正确凭证登录后进入现有 FreshPlate 主界面 |
| 会话保持 | 使用浏览器会话保存登录状态，刷新页面后保持登录 |
| 退出登录 | 顶部导航展示当前用户名并提供退出按钮，退出后返回登录页 |

**验收标准**：
- [ ] 未登录打开应用时默认显示登录页
- [ ] 用户名或密码为空时点击登录，页面显示必填错误
- [ ] 输入错误用户名或密码时，页面显示登录失败信息
- [ ] 输入正确预设用户名和密码后成功进入首页
- [ ] 刷新页面后保持登录状态
- [ ] 点击退出后返回登录页，并清除当前会话登录状态
